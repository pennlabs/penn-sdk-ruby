# PennSDK: Rubygem for Penn OpenData

This is the Penn OpenData API implementation in Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'penn_sdk'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install penn_sdk
